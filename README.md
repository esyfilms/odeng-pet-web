## Odeng Pet Web

This is the mobile web client for Odeng Pet V1.

Characteristics:
- static HTML/CSS/JS
- iPhone-first layout
- local persistence via `localStorage`
- suitable for GitHub Pages

### Run locally

Open [index.html](/Users/esyfilms/Desktop/codex_esyfilms/odeng-pet-web/index.html) in a browser, or serve the folder with any static file server.

### Notes

- This client mirrors the same V1 care loop as the Swift scaffold.
- It uses the shared sprite pack copied from `odeng-pet-shared`.
